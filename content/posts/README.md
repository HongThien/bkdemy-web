# Định dạng bài viết `/tin-tuc`

Mỗi bài = 1 file `.mdx` trong thư mục này (file này là `.md` nên KHÔNG bị liệt kê là bài viết).

```mdx
---
title: "Tiêu đề bài viết"
date: "2026-08-01"
excerpt: "1-2 câu tóm tắt hiện ở trang danh sách."
---

Nội dung bài viết ở đây, hỗ trợ Markdown/MDX đầy đủ.
```

Slug của bài = tên file (không đuôi `.mdx`). Ví dụ `khai-giang-2026.mdx` → `/tin-tuc/khai-giang-2026`.
